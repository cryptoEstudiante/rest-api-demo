Welcome to My Programming Demo!

I built the Web page you are viewing, and programmed the JavaScript client that informs the Web page.


My JavaScript client requests bitcoin price updates from two data providers: the CEX.IO bitcoin exchange and the SoChain blockchain reader. The blockchain reader reads the blockchain of the peer-to-peer (P2P) bitcoin network. The P2P bitcoin network is the network of machines running Bitcoin Core, the full reference client.

Note Please wait: My JavaScript client updates the chart every ten seconds, which meets data provider requirements. For this demo, data is being provided free of charge by each data provider.

Each data provider defines a Representational State Transfer (REST) API to enable subscribers, like my JavaScript client, to fetch price updates from a public endpoint. I coded my JavaScript client to implement an asynchronous Hypertext Transfer Protocol (HTTP) GET request to the endpoint of each provider. Data Provider responses are in JSON and my Javascript client converts the JSON to a Javascript object. The object provides access to the last bitcoin price sent by each data provider. My JavaScript client then displays the last prices using the Highcharts code libraries.

I developed my JavaScript client on a machine running Microsoft Windows 8.1. For debugging the client, I used the Google Chrome DevTools Console. Following is a summary of the programming tools and languages I used to create the demo:
  • NetBeans Integrated development environment (IDE) 8.2.
  • Cascading Style Sheets, level 2 (CSS2) and level 3 (CSS3) with media queries.
  • JQuery programming language.
  • JSON data interchange format.
  • Highcharts.
  • The latest Hypertext Markup Language (HTML) Living Standard, also known as HTML 5.
  • Blockchain SoChain API.
  • CEX.IO REST API.
